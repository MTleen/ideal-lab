# P2 大纲

## 文档信息
- 文档名称：ideal-lab Marketplace 与最佳实践介绍
- 大纲版本：V1.1（融入"从日常工作中沉淀"叙事线）
- 目标读者：事业部同事（技术+管理+产品混合背景）
- 预估总篇幅：7000-8000字

## 调研摘要

### 本地调研
从 ideal-lab 仓库中提取的关键信息：
- 仓库包含 9 个插件、42 个 Skill，覆盖开发、PPT、文档、知识库、调研等场景
- 四大工作流套件（dev-workflow / ppt-suite / document-workflow / knowledge-base）均采用"编排器 + Phase Skill"模式
- 每个工作流都支持手动模式和 YOLO（自动）模式
- ideal-document-workflow 的 12 阶段流程包含 6 个产物阶段和 5 个评审阶段，是最典型的编排器案例
- ideal-ralph 采用苏格拉底式任务澄清，体现"先想清楚再动手"的理念
- ideal-deep-research 提供 4 种深度模式（quick/standard/deep/ultradeep），体现"调研即能力"的设计
- 所有插件通过 Claude Code Plugin Marketplace 分发，一条命令安装
