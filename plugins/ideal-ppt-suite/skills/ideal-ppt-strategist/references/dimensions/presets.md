## 风格选择决策树（基于 ppt-master）

```
演示的核心诉求是什么？

Q1: 是否以图像/视觉展示/推广宣传为主？
├─ 是 → General Versatile（通用多能）
│   ├─ 品牌展示 → #4 极简风格 或 #9 杂志风格
│   ├─ 产品发布 → #3 商务风格 或 #11 动画风格
│   └─ 创意提案 → #5 手绘风格 或 #15 插画风格
└─ 否 ↓

Q2: 是否以数据分析/进度汇报为主？
├─ 是 → General Consulting（通用咨询）
│   ├─ KPI/仪表盘 → #8 块状风格 或 #10 图表风格
│   ├─ 综合报告 → #17 高密度信息图
│   └─ 进度汇报 → #3 商务风格
└─ 否 ↓

Q3: 是否以战略决策/说服影响为主？
├─ 是 → Top Consulting（顶级咨询）
│   ├─ 战略提案 → #3 商务风格
│   ├─ 研究报告 → #14 论文风格
│   └─ 高管汇报 → #4 极简风格
└─ 否 ↓

Q4: 特殊场景
├─ 技术架构 → #1 蓝图风格 或 #12 架构图风格
├─ 教育培训 → #2 黑板风格 或 #5 手绘风格
├─ 游戏/年轻 → #13 像素风格 或 #7 暗色风格
├─ 文艺/创意 → #6 水彩风格 或 #16 复古报纸风
└─ 科技前沿 → #7 暗色风格
```
