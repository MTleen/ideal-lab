# 图片纵横比与布局映射

## 核心原则

**布局容器的纵横比必须与图片原始比例匹配。** 不要将宽幅图片强行放入方形容器，也不要将竖版图片放入窄横条。

## 比例分类与推荐布局

| 纵横比 | 分类 | 推荐布局 | 容器尺寸参考 (1280x720 画布) |
|--------|------|---------|--------------------------|
| > 2.0 | 超宽幅 | 上下分割：图片全宽 | w=1200, h=400-500 |
| 1.5-2.0 | 宽幅 | 上下分割：图片顶部全宽 | w=1200, h=400-500 |
| 1.2-1.5 | 标准横版 | 左右分割：图片占一侧 | w=550-600, h=400-500 |
| 0.8-1.2 | 近似方形 | 左右分割：图片居中或一侧 | w=400-500, h=400-500 |
| < 0.8 | 竖版 | 左右分割：图片在左，文字在右 | w=350-400, h=550-600 |

## 布局模式详情

### 上下分割（宽幅图片）
```
┌─────────────────────────────┐
│                             │
│       图片 (全宽)            │
│                             │
├─────────────────────────────┤
│      文字内容区域            │
└─────────────────────────────┘
```
- 图片区域高度：≥ 300px
- 文字区域高度：≥ 150px
- 适用于 ratio > 1.5 的图片

### 左右分割（标准/竖版图片）
```
┌──────────────┬──────────────┐
│              │              │
│    图片       │   文字内容    │
│              │              │
│              │              │
└──────────────┴──────────────┘
```
- 图片侧宽度：≥ 350px
- 文字侧宽度：≥ 400px
- 适用于 ratio 0.5-1.5 的图片
- 竖版图片默认放左侧

### 图片居中（方形/通用）
```
┌─────────────────────────────┐
│        标题文字              │
├─────────────────────────────┤
│         ┌────────┐          │
│         │  图片   │          │
│         └────────┘          │
├─────────────────────────────┤
│        说明文字              │
└─────────────────────────────┘
```
- 图片最大占内容区域的 60%
- 适用于 ratio 0.8-1.2 的图片

## 引用图片的 SVG 语法

```xml
<!-- 标准引用 -->
<image href="../images/xxx.png" x="40" y="60" width="550" height="400"
       preserveAspectRatio="xMidYMid slice"/>

<!-- 占位符（图片尚未准备好时） -->
<rect x="40" y="60" width="550" height="400" fill="#F5F5F5"
      stroke="#CCCCCC" stroke-dasharray="8,4" rx="8"/>
<text x="315" y="260" text-anchor="middle" fill="#999999"
      font-size="14">图片占位：xxx.png</text>
```

## 尺寸处理规则

1. **不拉伸**：保持图片原始纵横比
2. **裁剪适配**：使用 `preserveAspectRatio="xMidYMid slice"` 居中裁剪
3. **留白填充**：使用 `preserveAspectRatio="xMidYMid meet"` 留白适配（当裁剪不可接受时）
4. **缩放**：图片最大边不超过容器尺寸
