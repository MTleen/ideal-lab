# 字体维度

标题与正文字体风格。

## 选项

| 选项 | 标题风格 | 正文风格 | 适用场景 |
|------|----------|----------|----------|
| `geometric` | 现代无衬线，干净棱角 | 干净无衬线 | 企业、科技、现代 |
| `humanist` | 友好无衬线，温暖曲线 | 易读无衬线 | 教育、普通受众 |
| `handwritten` | 马克笔/毛笔，有机质感 | 随性手写体或印刷体 | 创意、手绘、友好 |
| `editorial` | 粗衬线/无衬线混合，杂志风 | 经典衬线 | Keynote、杂志、高端 |
| `technical` | 等宽点缀，精确感 | 干净无衬线 | 开发者、数据、工程 |

## 渲染指南

### geometric

**标题**: 现代几何无衬线，干净棱角与一致的笔画宽度。参考 Futura、Avenir 或 Proxima Nova。粗体至半粗体。O、G 字符完美圆形。

**正文**: 为可读性优化的干净无衬线。常规字重。一致的 x 高度。充足的字间距。

**特征**：
- 字母形态中的数学精确
- 一致的笔画宽度
- 曲线中的完美几何
- 现代、权威的气场

### humanist

**标题**: 带有微妙笔画变化的友好无衬线。参考 Frutiger、Open Sans 或 Myriad。中粗至半粗体。温暖、亲和的字母形态。

**正文**: 易读的人文主义无衬线。舒适的行高。轻微的书法影响。

**特征**：
- 温暖、亲和的感受
- 微妙的笔画对比
- 开放的字腔确保可读性
- 自然、人文触感

### handwritten

**标题**: 粗体手写马克笔或毛笔字。厚重笔画带有机边缘。略微不齐的基线。渲染为实际手绘字母。

**正文**: 模仿笔记的清晰手写风格。随性但清晰。字母形态自然变化。

**特征**：
- 有机、不完美的字母形态
- 可见的笔刷/钢笔特征
- 随性、个人化感受
- 不是电脑字体——是实际绘制的字母

### editorial

**标题**: 粗衬线或高对比无衬线。杂志封面风格。戏剧性的尺度对比。参考 Playfair Display、Didot 或粗体压缩无衬线。

**正文**: 经典衬线用于长文本阅读。优雅、精致的字母形态。传统出版品质。

**特征**：
- 高对比（粗/细笔画）
- 戏剧性的标题
- 精致的气场
- 高端、出版品质

### technical

**标题**: 干净无衬线，带数据/代码用的等宽点缀。精确、工程化外观。参考 SF Mono 用于代码、Inter 用于标题。

**正文**: 为技术内容优化的干净无衬线。数字和代码用固定宽度。

**特征**：
- 数据元素用等宽字体
- 精确对齐
- 清晰的数字区分（0 vs O、1 vs l）
- 工程精确

## 字体渲染指令

由于图像生成器无法使用字体名称，需描述视觉特征：

| 选项 | 标题描述 | 正文描述 |
|------|----------|----------|
| geometric | "bold geometric sans-serif with perfect circular O shapes" | "clean modern sans-serif" |
| humanist | "friendly rounded sans-serif with warm letterforms" | "readable humanist sans-serif" |
| handwritten | "bold hand-drawn marker lettering with organic strokes" | "casual handwritten notes style" |
| editorial | "dramatic high-contrast serif with thick-thin stroke variation" | "elegant classic serif" |
| technical | "precise sans-serif with monospace numbers" | "technical sans-serif, fixed-width for code" |

## 组合注意事项

| 字体 | 最佳搭配 | 避免搭配 |
|------|----------|----------|
| geometric | clean 纹理、professional/neutral 色调 | organic 纹理 |
| humanist | organic/clean 纹理、warm 色调 | pixel 纹理 |
| handwritten | organic/paper 纹理、warm/vibrant 色调 | grid 纹理、professional 色调 |
| editorial | clean 纹理、vibrant/professional 色调 | pixel 纹理 |
| technical | grid/clean 纹理、cool/dark 色调 | paper 纹理、warm 色调 |
