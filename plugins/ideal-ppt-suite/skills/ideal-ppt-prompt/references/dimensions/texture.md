# 纹理维度

视觉纹理与背景处理。

## 选项

| 选项 | 背景 | 视觉元素 | 适用场景 |
|------|------|----------|----------|
| `clean` | 纯色，无纹理 | 干净线条、几何形状 | 高管、极简、企业 |
| `grid` | 微妙网格叠加 | 网格线、示意图、技术图表 | 技术、架构、工程 |
| `organic` | 柔和纹理、手绘质感 | 笔触、水彩、潦草线条 | 创意、教育、友好 |
| `pixel` | 大块像素、8-bit 美学 | 像素艺术、复古游戏元素 | 游戏、开发者、怀旧 |
| `paper` | 做旧/纹理纸张 | 复古元素、邮戳、磨损感 | 历史、遗产、叙事 |

## 渲染指南

### clean

- 纯色背景，无可见纹理
- 所有元素边缘锐利清晰
- 数字精确与清晰
- 最大化对比度确保可读性

### grid

- 浅色网格叠加（5-10% 不透明度）
- 工程图纸或蓝图质感
- 对齐辅助线可见但微妙
- 技术图纸美学

### organic

- 纸张颗粒或画布纹理
- 不完美的边缘、自然变化
- 手绘色彩填充
- 随性、亲和的感受

### pixel

- 可见像素网格（大块，非精细）
- 8-bit 色板美学
- 锯齿边缘（无平滑）
- 复古游戏 UI 元素

### paper

- 做旧纸张纹理（微妙的折痕、变色）
- 复古印刷痕迹
- 棕褐或暖色调
- 历史文档质感

## 组合注意事项

| 纹理 | 最佳搭配 | 避免搭配 |
|------|----------|----------|
| clean | professional、neutral 色调 | handwritten 字体 |
| grid | cool、professional 色调 | handwritten、vibrant 色调 |
| organic | warm、vibrant 色调 | technical 字体 |
| pixel | vibrant、dark 色调 | editorial 字体 |
| paper | warm 色调 | geometric 字体、minimal 密度 |
