# 色调维度

色温与配色风格。

## 选项

| 选项 | 色温 | 配色风格 | 适用场景 |
|------|------|----------|----------|
| `professional` | 冷中性 | 海军蓝、金色、结构化灰 | 商务、投资人、企业 |
| `warm` | 暖色 | 大地色系、橙色、自然色 | 教育、友好、亲和 |
| `cool` | 冷色 | 蓝色、灰色、青色、蓝绿 | 技术、数据、分析 |
| `vibrant` | 多变 | 高饱和、大胆色彩 | 营销、创意、吸睛 |
| `dark` | 暗色 | 深色背景配亮色点缀 | 娱乐、游戏、氛围感 |
| `neutral` | 中性 | 最少色彩、灰度为主 | 高管、极简、高级感 |

## 渲染指南

### professional

- 克制使用点缀色
- 金色仅用于强调
- 干净、机构化质感
- 均衡对比度

### warm

- 慷慨使用暖色调
- 自然、亲和的色彩
- 色彩之间柔和过渡
- 温馨氛围

### cool

- 蓝色主导色板
- 色彩使用体现技术精确
- 高对比度确保清晰
- 分析感、可信赖感

### vibrant

- 大胆的色彩组合
- 整体高饱和度
- 动态色彩对比
- 充满活力的视觉存在

### dark

- 深色背景主导
- 点缀色在暗底上跳跃
- 发光/发光效果
- 电影级氛围

### neutral

- 最少色彩使用
- 字体承载视觉重量
- 灰度层级
- 最大化高级感

## 组合注意事项

| 色调 | 最佳搭配 | 避免搭配 |
|------|----------|----------|
| professional | clean 纹理, geometric 字体 | organic 纹理, handwritten |
| warm | organic 纹理, humanist 字体 | pixel 纹理, minimal 密度 |
| cool | grid 纹理, technical 字体 | paper 纹理, handwritten |
| vibrant | pixel/organic 纹理, editorial 字体 | neutral 色调重叠 |
| dark | clean/pixel 纹理, technical 字体 | paper 纹理 |
| neutral | clean 纹理, geometric 字体 | organic 纹理, vibrant 元素 |
