# Notion风格

SaaS 仪表板美学，聚焦干净数据与生产力工具样式

## 设计美学

干净、功能性的 SaaS 界面美学。受仪表板启发的布局，清晰的数据层级。Notion、Linear 和现代生产力工具的样式。信息密集但组织有序。专业且值得信赖。

## 背景

- 颜色：浅灰 (#F7F7F5) 或纯白 (#FFFFFF)
- 纹理：无——干净的纯色背景

## 排版

### 主字体（标题）

系统 UI 字体栈或 Inter。半粗字重用于强调。干净、功能性的字母形态。略微紧凑的字间距。

### 副字体（正文）

同一家族常规字重。针对屏幕阅读优化。舒适的行高 (1.5-1.6)。

| 角色 | 颜色 | 色值 | 用途 |

| 背景 | 浅灰 | #F7F7F5 | 主背景 |
| 卡片背景 | 纯白 | #FFFFFF | 内容卡片 |

## 视觉元素

- 基于卡片的布局，带有 subtle 边框或阴影
- 干净的数据表格和图表
- 进度条和指标展示
- 基于图标的导航提示
- 复选框和开关样式
- 标签和标记芯片
- 微妙的悬停状态样式
- 面包屑和层级指示器

## 风格规则

### 应该做

- 使用基于卡片的内容组织
- 创建清晰的数据层级
- 使用 subtle 的阴影和边框
- 保持布局网格对齐
- 醒目地展示指标

### 不应该做

- 使用装饰性插画
- 添加渐变或复杂背景
- 创建艺术性布局
- 使用圆润的 blob 形状
- 添加幻灯片编号、页脚或标志

## 最适合用于

产品演示、SaaS 展示、生产力工具推介、指标仪表板、功能导览、B2B 演示、技术产品营销
