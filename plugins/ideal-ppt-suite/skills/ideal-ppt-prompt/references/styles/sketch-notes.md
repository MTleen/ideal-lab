# sketch-notes

柔和的手绘插画风格，清新精致，兼具极简编辑美学

## 设计美学

插画或手绘质感，笔触柔和放松。整体风格清新、精致，采用极简编辑手法。强调精准、清晰与知性优雅，同时优先传递温暖、亲切与友好感。

## 背景

- 颜色：暖调米白（#FAF8F0）
- 纹理：微妙的纸张颗粒感，略带暖调，避免冰冷感

## 字体

### 主字体（标题）

粗体手写马克笔字体或卡通海报字体。基线略有不齐，营造有机感。笔画粗壮，边缘柔和。以手绘字母呈现，非打字效果。

### 副字体（正文）

清晰的手写圆体或硬笔风格，模拟日常笔记。字号一致，带有轻微自然变化。以随意手写呈现，清晰但不呆板。

## 视觉元素

- 连接线带有手绘波浪感，不必完全笔直
- 概念抽象图标用于阐释思想，而非写实场景
- 色彩填充无需完全填满轮廓——保留手绘的随意感
- 圆角简单几何图形
- 箭头与指示符采用潦草、非正式风格
- 涂鸦式装饰元素：星星、螺旋线、下划线

## 风格规则

### 应当

- 保持版面开阔且结构清晰
- 强调信息层级与可读性
- 所有元素采用手绘质感
- 允许不完美——轻微抖动增添个性
- 元素层叠时保留微妙重叠

### 不应

- 使用完美的几何图形
- 创建写实元素
- 元素过度拥挤
- 使用纯白色背景
- 添加页码、页脚或徽标

## 最适用于

教育内容、知识分享、技术讲解、友好型演示、教程、入职材料
