# vector-illustration

扁平矢量插画风格，清晰的黑色描边搭配复古柔和配色

## 设计美学

扁平矢量插画，无渐变或 3D 效果。所有元素均有清晰、统一粗细的黑色描边。几何简化，将复杂物体还原为基础图形。玩具模型美学，可爱、俏皮、亲切友好。宽幅横向构图效果良好。

## 背景

- 颜色：奶油米白（#F5F0E6）
- 纹理：微妙的纸张纹理，温暖怀旧感，令人联想到复古印刷品

## 字体

### 主字体（标题）

大号粗体复古衬线体用于标题，传递权威与优雅。参考经典广告海报。字形干净，气场强大。

### 副字体（副标题）

全大写无衬线体，置于彩色矩形块内。标签式外观。与块色形成高对比。

### 正文字体

干净的几何无衬线体以保证可读性。类似 Futura、Avenir 等。字重始终一致。

## 视觉元素

- 所有物体均有闭合黑色描边（填色绘本风格）
- 圆角线端，避免尖锐棱角
- 树木简化为棒棒糖形或三角形
- 建筑简化为矩形块，配网格窗户
- 2.5D 透视（类等轴测但更自由）
- 通过层叠与重叠营造深度，而非大气透视
- 装饰几何元素：放射线（太阳光芒）、胶囊形云朵、圆点、星星
- 人物简化为简单几何图形，面部细节极少

## 风格规则

### 应当

- 始终保持一致的描边粗细
- 使用柔和、复古的配色
- 将所有物体简化为基础几何图形
- 通过层叠营造深度
- 添加俏皮的装饰元素

### 不应

- 使用渐变或写实阴影
- 创建写实元素
- 使用细线或变化不一的线宽
- 包含复杂精细的插画
- 添加页码、页脚或徽标

## 最适用于

教育演示、创意提案、儿童内容、品牌展示、温暖亲切的主题、讲解型内容
