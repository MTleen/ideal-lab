# 语言风格要求

## 1. 正式性要求

### 1.1 口语化词汇替换

技术文档应使用正式、规范的书面语言，避免口语化表达。

| ❌ 口语化 | ✅ 正式表述 |
|---------|-----------|
| 打通、整合 | 实现...互联互通 |
| 靠X补位 | 通过X进行补充 |
| 拼出 | 整合形成 |
| 站不住脚 | 缺乏依据 |
| 够用 | 满足需求 |
| 飘走了 | 扩散转移 |
| 算是 | 可视为 |
| 看起来简单 | 表面上较为简单 |
| 一天只能 | 单日仅能 |
| 早就 | 已 |
| 往往 | 常常/通常 |

### 1.2 比喻化表述替换

避免过度使用比喻，应采用准确的技术术语。

| ❌ 比喻化 | ✅ 准确表述 |
|---------|-----------|
| 数据孤岛 | 数据分散/系统独立 |
| 黄金30分钟 | 关键响应窗口期 |
| 通才 | 通用能力模型 |
| 专才 | 专业能力模型 |
| 孤岛 | 独立模块/独立服务 |
| 双引擎 | 双重驱动机制 |
| 锦上添花 | 辅助增强功能 |
| 刀刃上 | 关键环节 |
| 教科书 | 训练样本/标注数据 |

### 1.3 情感化表述替换

技术文档应保持客观、中立，避免情感色彩。

| ❌ 情感化 | ✅ 客观表述 |
|---------|-----------|
| 令人失望 | 未达预期 |
| 根本缺陷 | 核心局限 |
| 不得不加班 | 工作负荷超出正常范围 |
| 稍有疏漏就可能 | 存在遗漏风险 |
| 遗憾的是 | 经评估表明 |
| 遗憾 | 受限于 |
| 遗憾未能 | 由于...限制，未能 |

## 2. 准确性要求

### 2.1 数值表述

**必须包含的内容**：
- 具体数值或范围
- 数据来源或计算依据
- 适用条件说明

**示例对比**：

| ❌ 不够准确 | ✅ 准确表述 |
|-----------|-----------|
| 约2000条 | 2000条（基于历史数据统计，截至2025年12月） |
| 准确率约90% | 准确率达到88%-92%（基于测试集验证，n=1200） |
| 提升15个百分点 | 较基准值（65%）提升15个百分点 |
| 大量 | 超过100项 |
| 较高 | 达到X%以上 |

### 2.2 技术术语

**使用原则**：
- 首次出现时给出全称和缩写
- 同一术语全文保持一致
- 使用行业标准术语

**示例**：
```markdown
首次出现：采用消息队列遥测传输协议（Message Queuing Telemetry Transport，MQTT）
后续使用：MQTT协议
```

### 2.3 时间表述

| ❌ 模糊表述 | ✅ 精确表述 |
|-----------|-----------|
| 最近 | 近期（具体时间范围） |
| 很快 | 预计于X月内 |
| 长期 | 未来3-5年 |
| 短期 | X个月内 |
| 实时 | 毫秒级/秒级响应 |

## 3. 客观性要求

### 3.1 避免主观判断

| ❌ 主观表述 | ✅ 客观表述 |
|-----------|-----------|
| 这个方案很好 | 该方案在X方面具有优势 |
| 重要的是 | 关键因素包括 |
| 显然 | 经分析表明 |
| 毫无疑问 | 数据显示 |
| 理所当然 | 基于X原则 |

### 3.2 避免夸大表述

**禁止使用的表述**：
- 国际领先、国内首创、填补空白
- 颠覆性、革命性、突破性
- 完美、最优、最佳
- 零风险、零误差、100%准确

**可接受的替代表述**：
- 在X指标上达到Y水平
- 较现有方案提升Z%
- 在X场景下表现优异
- 风险可控、误差在可接受范围内

### 3.3 限定性表述

适当使用限定词，增强表述的严谨性：

| 类型 | 示例 |
|-----|------|
| 范围限定 | 在X条件下、针对Y场景、基于Z假设 |
| 程度限定 | 较为、相对、一定程度上 |
| 可能性限定 | 预计、预估、可能、通常 |

## 4. 句式结构要求

### 4.1 句子长度

- 单句不超过 40 字
- 复句不超过 80 字
- 超长句应拆分为多个短句

### 4.2 句式类型

**推荐使用的句式**：
- 陈述句（用于说明事实）
- 并列句（用于列举要点）
- 因果句（用于解释原因）

**避免使用的句式**：
- 疑问句（不应用于正文叙述）
- 感叹句（不应出现）
- 反问句（不应出现）

### 4.3 连接词使用

| 关系类型 | 推荐连接词 |
|---------|-----------|
| 递进 | 此外、同时、进一步 |
| 转折 | 然而、但需注意、值得关注的是 |
| 因果 | 因此、基于此、鉴于此 |
| 并列 | 首先...其次...最后、一方面...另一方面 |
| 总结 | 综上所述、总体而言、归纳起来 |
