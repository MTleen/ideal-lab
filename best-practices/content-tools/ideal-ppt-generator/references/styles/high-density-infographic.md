# 高密度信息图风格 (High-Density Infographic)

## 概述

专为高信息密度技术销售演示设计的学习风格。提炼自 JIMO PPT 的视觉模式，强调：
- 最小化留白，最大化信息展示
- 复合框 (Compound Boxes) 承载多层次信息
- 多列对比布局
- 流程图与管线可视化
- Badge 标签强调关键指标

## 视觉特征

### 布局原则
- 16:9 宽屏比例
- 纯白背景 (#FFFFFF)
- 紧凑的网格系统
- 最小化留白，最大化内容密度

### 核心视觉元素

#### 1. 复合框 (Compound Boxes)
带彩色标题栏的卡片框体：
```
┌─────────────────────────────────────┐
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │ ← 彩色标题栏 (8px)
├─────────────────────────────────────┤
│                                     │
│  内容区域                           │
│  - 多个要点                         │
│  - 嵌套信息                         │
│                                     │
└─────────────────────────────────────┘
```

#### 2. 多列对比布局
左右/上下对比结构：
```
┌─────────────────┬─────────────────┐
│  优势 (彩色)   │  劣势 (灰色)    │
├─────────────────┼─────────────────┤
│ ✓ 详细说明     │ ✗ 详细说明     │
│ • 要点1        │ • 要点1         │
│ • 要点2        │ • 要点2         │
└─────────────────┴─────────────────┘
```

#### 3. 流程管线 (Pipeline Flow)
横向箭头连接的处理阶段：
```
┌──────┐    ┌──────┐    ┌──────┐    ┌──────┐
│ 阶段1 │───→│ 阶段2 │───→│ 阶段3 │───→│ 输出  │
└──────┘    └──────┘    └──────┘    └──────┘
```

#### 4. Badge 标签
小型强调标签：
- 标签样式：`[标签文字]` 或 `● 标签`
- 用于：版本号、状态、关键指标

#### 5. 网格系统
- 6列/8列紧凑网格
- 用于：算子展示、特性矩阵

## 色彩系统

### 主色调
- Primary: 深藏青 (#1E3A5F)
- Accent: 蓝色 (#3B82F6)
- Success: 绿色 (#10B981)
- Warning: 橙色 (#F59E0B)
- Danger: 红色 (#EF4444)

### 背景色
- Card Background: #FFFFFF (白)
- Light Background: #F8FAFC (浅灰)
- Section Background: #EFF6FF (浅蓝)

### 文字色
- Primary Text: #1E3A5F (深藏青)
- Secondary Text: #64748B (灰色)
- White Text: #FFFFFF

## 排版规范

### 字体
- 标题: Bold Sans-Serif (如 Inter, Roboto)
- 正文: Clean Sans-Serif
- 紧凑行距: 1.2-1.4

### 字号层级
- 幻灯片标题: 28-32pt
- 卡片标题: 16-18pt
- 正文: 12-14pt
- 注释: 10-11pt

## 内容密度指南

### 高密度原则
1. **每页一个核心观点** + **3-5个支撑要点**
2. **使用缩写和符号** 减少文字量
3. **嵌套列表** 而非平铺
4. **图标辅助** 快速识别
5. **省略冠词和介词** 直接表述

### 视觉压缩技巧
- 使用 `•` 替代 `·`
- 使用 `→` 替代 `然后`
- 使用表格替代段落
- 使用网格展示多项内容

## 布局变体

### 封面布局
- 产品名称居中大字
- 标语突出显示
- 装饰线条分隔

### 目录布局
- 时间线/编号连接
- Badge 标签编号
- 左对齐紧凑列表

### 内容布局
- 2-3 列并排
- 卡片式分组
- 流程箭头连接

### 对比布局
- 左右分栏
- 彩色 vs 灰色
- ✓ vs ✗ 对比符号

### 管线布局
- 横向阶段流
- 垂直嵌套详细
- 输入 → 处理 → 输出

### 封底布局
- 居中产品信息
- 2x2 规格网格
- 突出标语横幅

## 适用场景

- 技术产品演示
- 销售提案
- 架构说明
- 案例研究
- 竞争优势对比
- 端到端流程展示

## 不适用场景

- 教学培训（需要解释）
- 领导汇报（需要简洁）
- 社交媒体（需要聚焦）
