# Dimension: Density (密度)

3 种密度选项，定义页面的信息密度和留白比例。

---

## 1. Minimal — 极简

**视觉描述**：
大量留白，信息极简。每页只呈现最核心的内容，通过空间感营造高端和聚焦的视觉效果。

**参数规格**：

| 参数 | 值 |
|------|-----|
| 每页要点数 | 1-2 个 |
| 正文基准字号 | 24px |
| 最大文字量 | 50 中文字符/页 |
| 留白比例 | ≥ 55% |
| 元素间距 | ≥ 48px |
| 卡片内边距 | ≥ 40px |
| 网格间距 (Gap) | 24px |

**排版特征**：
- 每页视觉焦点明确唯一
- 大量留白包围核心内容
- 标题字号大，正文字号小，对比强烈
- 图表和数据独立成页
- 极少使用边框和分割线

**适用场景**：
- 高管汇报
- 品牌展示
- 极简风格
- 主题演讲 (Keynote)
- 路演 (Pitch Deck)

**搭配建议**：
- Texture: clean / organic
- Mood: professional / neutral / dark
- Typography: geometric / editorial

**字号层级（基于 24px 基准）**：

| 元素 | 字号 | 行高 |
|------|------|------|
| Hero | 48-64px | 1.1 |
| H1 | 32-40px | 1.2 |
| H2 | 24-28px | 1.3 |
| Body | 16-18px | 1.6 |
| Caption | 12-14px | 1.5 |

---

## 2. Balanced — 平衡

**视觉描述**：
信息密度适中，留白与内容平衡。既有足够的信息传达，又保持视觉呼吸感。这是最通用的密度选项。

**参数规格**：

| 参数 | 值 |
|------|-----|
| 每页要点数 | 3-4 个 |
| 正文基准字号 | 20px |
| 最大文字量 | 100 中文字符/页 |
| 留白比例 | 35-45% |
| 元素间距 | ≥ 24px |
| 卡片内边距 | ≥ 24px |
| 网格间距 (Gap) | 16px |

**排版特征**：
- 每页传达 2-4 个要点
- 适度的留白保持视觉舒适
- Bento Grid 布局的理想密度
- 图文混排和谐
- 适合大多数商务场景

**适用场景**：
- 通用商务汇报
- 项目进度汇报
- 竞品分析
- 产品介绍
- 大多数常规演示

**搭配建议**：
- Texture: clean / grid / paper
- Mood: professional / warm / cool / neutral
- Typography: geometric / humanist / editorial

**字号层级（基于 20px 基准）**：

| 元素 | 字号 | 行高 |
|------|------|------|
| Hero | 40-52px | 1.15 |
| H1 | 28-36px | 1.25 |
| H2 | 20-24px | 1.35 |
| Body | 16-18px | 1.6 |
| Caption | 12-14px | 1.5 |

---

## 3. Dense — 高密

**视觉描述**：
信息密度高，页面利用率最大化。在有限页面内传达大量信息，适合数据密集和专业受众的场景。

**参数规格**：

| 参数 | 值 |
|------|-----|
| 每页要点数 | 5-7 个 |
| 正文基准字号 | 18px |
| 最大文字量 | 150 中文字符/页 |
| 留白比例 | 20-30% |
| 元素间距 | ≥ 12px |
| 卡片内边距 | ≥ 16px |
| 网格间距 (Gap) | 8-12px |

**排版特征**：
- 每页承载大量信息
- 留白最小化但不消失
- 多卡片/多区域布局
- 小字号、紧凑间距
- 适合信息图和仪表盘

**适用场景**：
- 高密度信息图
- 数据仪表盘
- 技术文档
- 综合分析报告
- 竞品全面对比
- 专家级受众演示

**搭配建议**：
- Texture: grid
- Mood: professional / cool
- Typography: geometric / technical

**字号层级（基于 18px 基准）**：

| 元素 | 字号 | 行高 |
|------|------|------|
| Hero | 36-44px | 1.15 |
| H1 | 24-30px | 1.2 |
| H2 | 18-22px | 1.3 |
| Body | 14-16px | 1.5 |
| Caption | 10-12px | 1.4 |

---

## 密度选择决策树

```
受众是谁？
├─ 高管/决策者
│   → Minimal（结论先行，信息聚焦）
├─ 通用/混合受众
│   → Balanced（信息充足，视觉舒适）
└─ 专家/从业者
    → Dense（信息完整，数据密集）

内容类型？
├─ 品牌展示/路演
│   → Minimal
├─ 常规汇报/分析
│   → Balanced
└─ 数据报告/技术文档
    → Dense

页数限制？
├─ 页数充裕（每主题 ≥ 2 页）
│   → Minimal 或 Balanced
├─ 页数适中
│   → Balanced
└─ 页数紧张（信息多页数少）
    → Dense
```
