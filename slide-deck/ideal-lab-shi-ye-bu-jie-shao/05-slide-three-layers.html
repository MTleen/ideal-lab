Plugin → Skill → References 三层架构
Plugin（插件）
最大的能力单元，每个 Plugin 对应一个独立工作领域
ideal-dev-workflow ideal-ppt-suite ideal-document-workflow
Plugin 之间完全解耦，按需安装，不需要全量部署
包含
↓
Skill（技能）
Plugin 内的执行单元，每个 Skill 做一件事
通过 SKILL.md 定义"做什么"和"怎么做"
一个 Plugin 下多个 Skill 按流程编排，形成完整工作流
读取
↓
References / Scripts
references/ — 参考文档（写作规范、模板、技术规格）
scripts/ — 执行脚本（格式校验、文件转换、自动化工具）
Skill 执行时读取这些资源，确保输出符合预设标准
透明可审计
  • 任何 Skill 的行为通过阅读 SKILL.md 完整了解
  • 不需要逆向工程
  • 版本可追溯，变更可对比
职责分离
  • Plugin 管领域边界
  • Skill 管具体执行
  • References 管标准资源
  • 三层独立迭代，互不影响
