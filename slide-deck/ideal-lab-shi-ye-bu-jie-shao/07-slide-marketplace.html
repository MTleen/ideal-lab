Marketplace 三步安装
第一步：添加 Marketplace 源
claude plugin marketplace add https://github.com/MTleen/ideal-lab
  • 告诉 Claude Code 从哪里获取插件清单和安装包
  • 只需执行一次
→
第二步：按需安装插件
# 文档写作工作流（8 个 Skill） claude plugin install ideal-document-workflow@ideal-lab # 深度调研工具 claude plugin install ideal-deep-research@ideal-lab # PPT 全链路（9 个 Skill） claude plugin install ideal-ppt-suite@ideal-lab
  • 每个插件独立安装，用哪个装哪个
  • 不需要全量安装
→
第三步：斜杠命令调用
/ideal-document-workflow:ideal-document-workflow /ideal-deep-research:deep-research /ideal-ppt-suite:ideal-ppt-workflow
  • 安装后 Skill 自动加载
  • 调用编排器即可自动走完整个流程
每个工作流插件都有编排器 Skill，调用编排器即可自动走完整个流程，无需逐阶段手动触发。
