# PPT Prompt - Slide 01: Cover

=== Style Brief ===
Visual style: Professional technology presentation, blue-themed, clean card-based layout
Color palette: Primary #1A56DB(telecom blue) / Secondary #7C3AED(tech purple) / Accent #F59E0B(data orange)
Module colors: Red=#DC2626(core/AI) Blue=#2563EB(cloud) Orange=#D97706(users) Green=#059669(data)
Typography: Clear hierarchy - titles bold/dark, body readable, numbers large
Font: PingFang SC, Microsoft YaHei, Noto Sans SC, sans-serif
Density: dense - pack data, use tables/cards/architecture diagrams, minimize whitespace
Canvas: 1280x720 (16:9)
Language: Chinese

=== Slide Type ===
Cover slide - gradient or solid dark background

=== Content ===

主标题（超大号、白色、粗体、居中）：
云端技术助力人工智能高效落地

副标题（中号、浅色、居中、主标题下方）：
从算力困境到云智融合的未来

底部信息行（小号、浅灰色）：
- 演讲人：[演讲人姓名]
- 日期：2026年
- 单位：[组织名称]

装饰元素（3个徽章/badge，水平排列于副标题与底部信息之间）：
1. 蓝色徽章：☁️ 云计算
2. 紫色徽章：🧠 人工智能
3. 橙色徽章：🔗 云智融合

背景：深色渐变（从 #0F172A 到 #1E3A5F），可加入细微的科技网格纹理或粒子光效。

=== Layout Guidance ===
- 主标题居中偏上，字号最大，白色加粗
- 副标题紧跟主标题下方，字号约为主标题的40%
- 3个徽章水平居中排列，间距均匀
- 底部信息行左对齐或居中，小字号
- 整体视觉重心居中，营造科技感与专业感
