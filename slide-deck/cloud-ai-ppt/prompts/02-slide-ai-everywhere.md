# PPT Prompt - Slide 02: AI is Everywhere

=== Style Brief ===
Visual style: Professional technology presentation, blue-themed, clean card-based layout
Color palette: Primary #1A56DB(telecom blue) / Secondary #7C3AED(tech purple) / Accent #F59E0B(data orange)
Module colors: Red=#DC2626(core/AI) Blue=#2563EB(cloud) Orange=#D97706(users) Green=#059669(data)
Typography: Clear hierarchy - titles bold/dark, body readable, numbers large
Font: PingFang SC, Microsoft YaHei, Noto Sans SC, sans-serif
Density: dense - pack data, use tables/cards/architecture diagrams, minimize whitespace
Canvas: 1280x720 (16:9)
Language: Chinese

=== Slide Type ===
Content slide - three scenario cards + core question

=== Content ===

标题（顶部，粗体）：
你的生活中，AI已经无处不在

时间线提示（标题下方醒目标注）：
⏱️ 以上这些，全部发生在一个小时之内

三大日常场景卡片（水平排列，每个卡片包含图标+标题+描述）：

卡片1 - 智能闹钟（使用橙色 #D97706 边框或图标）
- 标题：🌅 智能闹钟
- 描述：根据你的睡眠状态，在最合适的时机叫醒你
- 细节：监测睡眠周期 → 自动调整唤醒时间 → 不再被硬叫醒

卡片2 - 视频推荐（使用蓝色 #2563EB 边框或图标）
- 标题：🎬 视频推荐
- 描述：打开手机，它已经知道你想看什么
- 细节：学习你的兴趣偏好 → 精准推送内容 → 越看越懂你

卡片3 - AI客服（使用紫色 #7C3AED 边框或图标）
- 标题：🎧 AI客服
- 描述：刚才遇到的问题，3分钟就被AI解决了
- 细节：自动理解问题 → 智能匹配方案 → 快速解决

核心追问（底部醒目大字，使用红色 #DC2626 或强调色）：
❓ 是什么在驱动这些AI？
❓ 它们需要多大的"能量"？

=== Layout Guidance ===
- 三张卡片等宽水平排列，每张卡片包含图标、标题、描述文字
- 卡片之间留有均匀间距
- 时间线提示放在标题下方，使用醒目但不突兀的样式
- 底部核心追问使用较大字号和强调色，引导观众思考
- 整体布局紧凑，信息密度高
