## Headline
AI智能体——从"工具"变成"助手"

## Subtitle / Context
AI正在经历一次根本性的角色转变：从"你按一下它动一下"的计算器，变成"你交代目标，它自己跑起来"的实习生。这就是AI智能体（AI Agent）。

## Core Content

### 类比：计算器 vs 实习生

**传统AI = 计算器**
- 你按一下，它动一下
- 每一步都需要人类指令
- 只能执行单一任务
- 示例：你问"今天天气如何"，它回答"25度，晴天"

**AI智能体 = 实习生**
- 你交代一个目标，它自己规划步骤、自己执行
- 不需要你事无巨细地指挥
- 能完成多步骤、多环节的复杂任务
- 示例：你说"帮我安排明天出差"，它自己查航班、订酒店、设闹钟、通知同事

### AI智能体三大特征

**1. 主动性（Proactivity）**
- 不等待指令，主动发现问题、提出建议
- 示例：网络出现异常，智能体主动检测并预警

**2. 规划能力（Planning）**
- 接收目标后，自动拆解为多个子任务
- 示例："处理客户投诉" → 分析问题 → 查询知识库 → 生成回复方案 → 执行

**3. 多步骤执行（Multi-step Execution）**
- 连续完成一系列动作，形成完整工作流
- 示例：客服智能体同时完成"理解问题 → 查询系统 → 办理业务 → 确认结果"

### 中国电信实践

**350+ AI智能体已部署**
- 覆盖领域：客户服务、网络运维、市场营销、运营管理
- 客服场景：智能体理解用户意图，自动查询并办理业务
- 运维场景：智能体监控网络状态，自动定位并修复故障
- 营销场景：智能体分析用户画像，自动推荐个性化方案
